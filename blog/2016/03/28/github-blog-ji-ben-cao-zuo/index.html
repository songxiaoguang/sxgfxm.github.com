Github Blog - 基本操作

Mar 28th, 2016 7:43 pm

创建新博文

rake new_post["article_name"]
会在octopress/source/_post/目录生成.markdown文件，可使用markdownpad2编辑。

删除博文

只需删除对应的markdown文件即可。

更新博客

rake generate
自动生成/public目录下的展示页面。

rake preview
本地预览，地址为http://localhost:4000，ctrl+C终止预览。

rake deploy

push到github

git add .
git commit -m "message"
git push origin source
